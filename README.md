mackerel-plugin-http
=====================

## Usage

```shell
$ mackerel-plugin-http -url "http://example.com"
```

## Example of mackerel-agent.conf

```
[plugin.metrics.http]
command = "/path/to/mackerel-plugin-http -url \"http://example.com\""
```
